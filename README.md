# 📡 dLaser Terminal - deze OS Project 🇩🇿

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)
[![Platform: MicroPython](https://img.shields.io/badge/Platform-MicroPython-red.svg)](https://micropython.org/)

**dLaser Terminal** هو نظام استقبال بيانات لاسلكي عبر نبضات الليزر، مصمم ليعمل كجزء من نظام التشغيل **deze OS**. تم تطويره واختباره بنجاح في مخبر الإعلام الآلي بمتوسطة تورة (سوق أهراس).

## 🚀 نظرة عامة (Overview)
يعتمد النظام على تحويل نبضات الضوء المستقبلة عبر حساس LDR إلى بيانات رقمية (Binary) ثم معالجتها وتحويلها إلى نصوص ASCII تُعرض بدقة عالية على شاشة ILI9341 باستخدام تقنية الـ FrameBuffer لضمان السرعة والوضوح التام أثناء الاستقبال اللحظي.

## 🛠️ المكونات (Hardware)
* **المعالج:** Raspberry Pi Pico (RP2040).
* **الشاشة:** TFT 2.4 inch (ILI9341) SPI.
* **الحساس:** خلية ضوئية (LDR) موصلة بمنفذ ADC0.
* **التنبيه:** بايزر (Buzzer) للمؤثرات الصوتية التفاعلية.
* **الإرسال:** نظام Arduino Uno مزود بوحدة ليزر (650nm).

## 🔌 التوصيلات (Pinout)
| المكون | منفذ الشاشة | منفذ البيكو (GP) | الوظيفة |
| :--- | :--- | :--- | :--- |
| **ILI9341** | CS | GP17 | اختيار الشاشة |
| **ILI9341** | SCK | GP18 | نبضات المزامنة |
| **ILI9341** | MOSI | GP19 | نقل البيانات |
| **ILI9341** | DC | GP20 | تحديد الأوامر |
| **ILI9341** | RST | GP21 | إعادة التشغيل |
| **LDR** | Signal | **GP26 (ADC0)** | استقبال الليزر |
| **Buzzer** | Signal | **GP15 (PWM)** | المؤثرات الصوتية |

## 📜 بروتوكول الإرسال (Protocol)
يستخدم النظام ترميز 8-بت (Standard ASCII):
* **سرعة البت:** 50ms (قابلة للضبط).
* **ترتيب البتات:** MSB First (البت الأكبر أولاً).
* **نظام الصوت:** * نغمة 1800Hz عند استقبال حرف.
  * نغمة 400Hz عند استقبال مسافة.
  * نغمة مزدوجة عند حدوث خطأ في الإشارة.
